# Product Requirements Document (PRD)

## 1. Project Overview
**LitLens AI** is an AI-powered literature survey and research assistant designed to streamline academic research. It helps users upload research papers (PDFs), extract key information (methodology, datasets, findings), and synthesize literature reviews efficiently via a unified Intelligence Hub.

## 2. Goals & Objectives
- **Core Goal**: Build an AI research assistant that optimizes literature reviews.
- **Current Status**: MVP Successfully Launched with Enterprise-grade AI Orchestration.

## 3. Target Audience
- University students (undergraduates and graduates)
- Academic researchers
- Anyone needing to synthesize multiple technical documents

## 4. Core Features (MVP - All Implemented)
1. **Document Management**: Securely upload and manage research papers (PDFs) with Inngest background processing.
2. **AI Summarization**: Automatically generate summaries of uploaded papers via OpenAI.
3. **Research Intelligence Hub**: A unified workspace featuring timelines, live project health, and insight generation.
4. **Targeted Extraction**: Extract methodology, datasets, and key findings from papers dynamically.
5. **Multi-Paper Comparison**: Compare multiple papers to identify common themes and contrasting results.
6. **Research Gap Identification**: AI-assisted detection of gaps in the current literature.
7. **Literature Review Generation**: Draft initial literature reviews based on selected papers with embedded citations.

## 5. Non-Goals (Out of Scope for MVP)
- Real-time collaborative editing (Google Docs style)
- Direct integration with institutional paywalls/libraries
- Massive scale distributed processing

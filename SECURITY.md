# Security Policy

## Supported Versions

Only the latest version of LitLens AI (`v1.x.x`) is currently supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability within LitLens AI, please DO NOT open a public issue.
Instead, please send an e-mail to the maintainers securely. 

We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress. If a fix is accepted, we will coordinate the public disclosure with you to ensure users have time to patch their instances.

## AI Prompt Injection

LitLens AI utilizes Large Language Models (LLMs). While we have strict system prompts, prompt injection is a known industry risk. If you find a severe prompt injection vulnerability that exposes database information or internal system secrets, please report it via the responsible disclosure email above.
